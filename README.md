# Network Traffic Protocol Fingerprinting (Task 28)

## 1. Project Overview
This project focuses on classifying encrypted network flows into application protocols using machine learning techniques.

## 2. Dataset
- **Name:** ISCX VPN-nonVPN Dataset (University of New Brunswick).
- **Description:** A dataset containing various types of traffic like HTTP/S, DNS, VoIP, etc.

## 3. Technology Stack
- **Language:** Python.
- **Libraries/Tools:** Scikit-learn, XGBoost, or Random Forest.
- **Environment:** Google Colab / Jupyter Notebook.

## 4. Expected Results
- **Protocol Classification Accuracy:** > 92%
- **Macro F1-score:** > 0.88
